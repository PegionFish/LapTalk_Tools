# HWiNFO CSV 折线图导出工具

一个面向 Windows GUI 的小工具，用来读取 HWiNFO 导出的 CSV 文件，按时间戳绘制指定参数的折线图，并导出透明背景 PNG，方便接入后续开发流程。

## 功能

- 读取 HWiNFO CSV 文件
- 以 `Date + Time` 组合出的时间戳作为 X 轴
- 以指定参数的数值作为 Y 轴
- 支持多选参数并在同一张图里叠加显示
- 导出透明背景 PNG
- 自动预览图表，无需手动点击刷新
- 选择 CSV 后会预解析全部数值列到内存中，后续筛选与绘图更流畅
- 支持标题、线宽、网格、图例位置、时间刻度密度滑块，以及不同参数的独立颜色设置
- 支持中文列名、重复列名和列名筛选，并自动识别常见 CSV 编码

## 环境

- Windows
- Python 3.10+
- `matplotlib`

安装依赖：

```powershell
pip install -r requirements.txt
```

## 启动

双击 `run_gui.bat`，或者手动执行：

```powershell
python .\main.pyw
```

## 使用方式

1. 点击“浏览...”选择 HWiNFO CSV 文件
2. 在左侧筛选并多选要绘制的参数
3. 按需设置标题、线宽、时间刻度密度、固定时间间隔、图例、输出宽高和 DPI，并可为不同参数指定颜色
4. 图表会在选择参数或调整设置后自动预览
5. 点击“导出透明 PNG”

## 说明

- 图表背景为透明，但坐标轴、文字、网格线和曲线会保留
- 预览会在后台线程中自动生成，并按窗口大小自适应缩放显示
- 预览会保持你填写的宽高比与 DPI 缩放关系，避免显示比例失真
- 导出 PNG 仍然使用你填写的完整宽高与 DPI，预览只在展示阶段自适应缩放
- X/Y 轴标题输入已移除，时间轴右下角的额外日期偏移标注也已去掉
- 可以通过滑块调节时间刻度密度；如果填写了固定时间间隔（秒/分钟/小时），则会优先按固定间隔显示
- 时间轴刻度会比之前更密，短时间范围下会优先显示更细的分钟级刻度
- 如果 CSV 中存在重名列，界面会显示 CSV 列号和重复序号来区分
- 如果 CSV 使用 UTF-8、UTF-16、GB18030/GBK 等常见编码，工具会自动选择更合理的解码方式，减少中文乱码
- 当前仓库已适配样例文件 `R23-15.CSV`
