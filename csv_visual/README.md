# HWiNFO CSV 折线图导出工具

一个面向 Windows GUI 的小工具，用来读取 HWiNFO 导出的 CSV 文件，按时间轴绘制指定参数的折线图，并导出透明背景 PNG，方便接入后续剪辑、排版或可视化流程。

## 功能

- 支持一次添加多个 HWiNFO CSV 文件，也支持直接将多个 CSV 拖入窗口
- 支持移除选中文件、清空全部文件，并自动按规范化绝对路径去重
- 支持四区布局：文件管理、图表预览、参数与图表设置、统一时间编辑工作台
- 支持设置“基准文件”，并对每个文件分别输入正负偏移量或拖动时间轴片段进行手动时间对齐
- 支持在统一时间轴中拖动 CSV 片段左右边缘，裁剪单个 CSV 的有效片段
- 支持在统一时间轴中拖动全局工作区边界，裁剪最终预览 / 导出范围
- 支持在同一张图中同时选择不同文件、同名或异名参数进行叠加对比
- 支持按 `SeriesKey(session_id, column_index)` 维度为不同文件中的参数独立设置颜色
- 支持对齐后的全局时间轴裁剪，允许出现负时间，并与单文件有效片段裁剪同时生效
- 支持后台自动预览与后台数值序列预载，导出 PNG 与预览共用同一套渲染参数
- 支持标题、字体、线宽、图例位置、纯曲线模式、坐标轴/网格/文字颜色、时间刻度密度、固定时间间隔等样式设置
- 支持中文列名、重复列名、列名筛选，以及 UTF-8 / UTF-16 / GB18030 / GBK 等常见编码自动识别

## 环境

- Windows
- Python 3.10+
- `matplotlib`

安装依赖：

```powershell
pip install -r requirements.txt
```

## 启动

双击 `run_gui.bat`，或者手动执行：

```powershell
python .\main.pyw
```

## 使用方式

1. 点击“添加 CSV...”选择一个或多个 HWiNFO CSV 文件，或直接把多个 CSV 拖入窗口
2. 在“模块 1 · 文件管理”查看已加载文件，必要时移除、清空或切换基准文件
3. 选中文件后可在模块 1 编辑：
   - `别名`：用于参数列表和图例显示
   - `偏移(秒)`：正值向右移动曲线，负值向左移动曲线
   - `-10s / -1s / +1s / +10s`：快速微调时间对齐
4. 在“模块 3 · 参数筛选 + 图表设置”中筛选并选择要绘制的参数，并按需设置标题、字体、线宽、图例、时间刻度和颜色
5. 在“模块 4 · 可视化范围 + 时间轴”中完成时间编辑：
   - 拖动片段主体：调整该 CSV 的 `offset_seconds`
   - 拖动片段左右边缘：调整该 CSV 的原始时间有效片段
   - 拖动顶部工作区左右边界：调整最终预览 / 导出范围
   - 默认吸附到 `1s`，按住 `Shift` 拖动可吸附到 `0.1s`
6. “模块 2 · 图表预览”会自动在后台刷新，确认后点击“导出透明 PNG”

## 多文件对比说明

- 首个导入文件默认成为基准文件，基准切换时会自动重归一化偏移量，避免当前可视布局跳变
- 对齐公式为：

```text
aligned_elapsed_seconds = raw_elapsed_seconds + offset_seconds
```

- 单个 CSV 有效片段基于“原始 CSV 时间”裁剪：

```text
source_trim_start_seconds <= raw_elapsed_seconds <= source_trim_end_seconds
```

- 全局工作区基于“对齐后的时间轴”裁剪：

```text
work_area_start_seconds <= aligned_elapsed_seconds <= work_area_end_seconds
```

- 最终参与绘图的数据点必须同时落在该 CSV 的有效片段内，并落在全局工作区内
- 当某个文件需要向左对齐时，时间轴允许显示负值
- 多文件模式下，参数列表显示为 `[文件别名] [列号] 参数名`，图例会自动包含文件别名以避免混淆
- 文件表格中的“有效片段”显示该 CSV 当前参与比较的原始时间段；顶部工作区则显示最终预览 / 导出的全局时间段

## 颜色与预览

- 参数颜色支持直接输入 `66CCFF` / `#66CCFF`，也支持使用取色器
- 坐标轴、网格、时间文字、数值文字、图例文字都可以独立设置颜色
- 预览在后台线程中按当前宽度、高度、DPI 和样式生成透明 PNG，再在窗口中自适应显示
- 拖动时间轴时会节流刷新预览，释放鼠标后会立即提交一次最终预览
- 导出 PNG 与预览共用同一套样式、颜色、CSV 有效片段和工作区配置

## 打包 EXE

1. 确保已安装 `requirements.txt` 中的依赖，其中已包含打包所需的 `PyInstaller`
2. 执行：

```powershell
python .\build_exe.py
```

或使用 PowerShell 包装脚本：

```powershell
.\build_exe.ps1
```

3. 默认产物位于：

```text
csv_visual\dist\HWiNFO-CSV-Plotter.exe
```

4. 如需生成目录模式而不是单文件 EXE，可执行：

```powershell
python .\build_exe.py --onedir
```

5. 如只想查看将要执行的 PyInstaller 命令，可执行：

```powershell
python .\build_exe.py --dry-run
```

## 说明

- 图表背景为透明，但坐标轴、文字、网格线和曲线会保留
- 预览区会按窗口大小自动等比 fit，不提供滚动查看原图细节
- 纯曲线模式会关闭网格、图例、时间文字和数值文字，只保留数据曲线
- 时间刻度会在保证可读性的前提下尽量细化；若填写固定时间间隔，则优先使用固定间隔
- 时间轴中的“重置所选对齐”会把所选文件偏移归零；“重置所选裁剪”会恢复所选 CSV 全长参与比较；“重置工作区”会把全局工作区恢复到当前有效内容范围
- 若需要手工验证，建议使用你自己的 HWiNFO CSV 导出文件，避免把真实数据固化到自动化测试中
